# EEG Signal Analysis using Python

## Overview
This project performs preprocessing and frequency-domain analysis of EEG signals using the MNE library.

## Objectives
- Load EEG dataset from PhysioNet
- Apply band-pass filtering (1–40 Hz)
- Extract Alpha (8–12 Hz) and Beta (13–30 Hz) bands
- Compute Power Spectral Density (PSD)
- Calculate average band power

## Tools Used
- Python
- MNE
- NumPy
- Matplotlib

## Applications
This analysis provides foundational techniques used in Brain–Computer Interface (BCI) and neuroengineering research.
